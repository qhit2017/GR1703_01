1 面向对象的三大特征是什么，分别有什么好处？
	封装(){
		隐藏类的具体实现
		避免不合理的操作
		}

	继承(){
		节省重复代码量
		}
	多态(){
		增加了对象的表现形式
		}

2 面向对象的三大特征，分别是如何实现的？
	封装（）{
		将属性私有，然后提供属性的get，set方法
		}
	继承（）{
		子类名 +extends关键词+父类名
		}
	多态（）{
		通过方法的重写和重载来实现
		}
4 总结下抽象类与接口有什么区别？
     1 抽象类可以有抽象方法和普通方法，接口只能有抽象方法；
     2 抽象类只能单继承，接口可以多实现；
