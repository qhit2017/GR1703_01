标识符

 定义:用来指示某个实体的符号。
规范:  1.由大小写英文字母、下划线、$组成；
          2.不能以数字开头；
          3.不能是Java中的保留的关键字；
          4.严格区分大小写；
          5.命令最好反应其作用，做到见名知意。

类
规范:不能是中文，是大写字母开头，以驼峰方式来命名。

变量
概念:可以变化的量；
规范:首字母小写，不能是中文，以驼峰方式命名；
语法:数据类型   变量名   =   数值;

数据类型                          位数
byte                            1个字节8位
short(短整型)              2个字节16位
int(整形)                     4个字节32位
long(长整型)               8个字节64位
float                           4个字节32位
double                       8个字节64位
char(字母)                   2个字节16位
bolean                        1个字节8位
Java中的小数默认为double类型；float类型，需在变量后加上f；
整数型通常用int，浮点型常用double。

算数运算符:+，-，*，/，%；
关系运算符:>,<,>=,<=,==,!=;
逻辑运算符:&&(与),||(或),!(非);

与(&&):只要有一个结果false,结果就是false;只有两边都是true时,结果才是true;
或(||):只要有一个结果是true,结果就是true;只有两边都是false时,结果才是false;
非(!):与原来表达式相反;
